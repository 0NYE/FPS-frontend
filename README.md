# FPS(Frontend Practice Service)

> 클론코딩을 재미있고 효과적이게

- FPS는 정답과 제출물간 이미지 유사도 수치를 실시간으로 제공하여 학습자의 흥미를 유발합니다.
- FPS는 lighthouse 검사 결과를 제공하여 학습자가 스스로 피드백할 수 있도록 돕습니다.
